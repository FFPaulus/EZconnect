#Instructions:
1. start vnc server: ```x11vnc -forever```
2. start websockify: ```websockify 3456 127.0.0.1:5900```
3. go to ```127.0.0.1:9292``` and log in
4. change settings to host 127.0.0.1, port 3456
5. click connect
